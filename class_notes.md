## 5-5 내장함수
 - abs : 절댓값을 돌려주는 함수
 - input : 사용자 입력을 받는 함수
 - len : 입력값의 길이를 돌려주는 함수
 - map : 함수와 반복가능한 자료형을 입력받고 결과를 묶어서 반환하는 함수
 - zip : 동일한 갯수로 이루어진 자료형을 묶어주는 함수
